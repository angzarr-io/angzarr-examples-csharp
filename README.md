# angzarr-examples-csharp

Example implementations demonstrating Angzarr event sourcing patterns in Csharp.

## Prerequisites

- Csharp build tools
- Buf CLI for proto generation
- Kind (for Kubernetes deployment)

## Building

See individual component directories for build instructions.

## Running

### Standalone Mode

Run with standalone runtime configuration.

### Kubernetes Mode

```bash
skaffold run
```
